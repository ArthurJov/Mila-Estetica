# 💆‍♀️ Site Profissional de Estética

Este projeto é um site desenvolvido para uma esteticista profissional, com foco em apresentar serviços, fortalecer presença digital e facilitar o contato com clientes.

🔗 **Acesse o site:**  
👉 [ludmilapaixao.unaux.com](https://ludmilapaixao.unaux.com/)

---

## 🧾 Sobre o Projeto

O site foi criado para atender às necessidades de uma profissional da área de estética, oferecendo uma experiência moderna, elegante e otimizada para dispositivos móveis.

A proprietária do projeto é **Ludmila Paixão**, esteticista, e o site foi pensado para transmitir:

- Sofisticação
- Confiança
- Bem-estar
- Profissionalismo

---

## 🎯 Objetivos

- Apresentar os serviços de forma clara e atrativa  
- Facilitar o agendamento de clientes  
- Criar uma presença digital profissional  
- Melhorar a experiência do usuário (UX) principalmente no mobile  
- Servir como vitrine online da profissional  

---

## 🧩 Funcionalidades

- 📱 Design responsivo (mobile-first)
- 📖 Seção de serviços detalhados
- 💬 Botão de contato direto via WhatsApp
- ⭐ Depoimentos de clientes
- 🖼️ Galeria de imagens
- ❓ FAQ (Perguntas Frequentes)
- 📍 Navegação por scroll suave entre seções
- 🍔 Menu mobile otimizado (hamburger menu)

---

## 🎨 UI/UX

O design segue uma linha:

- Minimalista
- Elegante
- Tons suaves e sofisticados
- Foco em legibilidade e clareza
- Navegação intuitiva

Além disso:
- Prioridade total para experiência mobile
- Elementos bem espaçados
- Hierarquia visual clara

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 📱 Responsividade

O site foi desenvolvido com abordagem **mobile-first**, garantindo:

- Excelente usabilidade em smartphones  
- Adaptação fluida para tablets e desktops  
- Componentes otimizados para toque
